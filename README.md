# 🌽 基于1D-CNN的玉米近红外光谱(NIR)成分定量分析

本项目利用深度学习技术，通过近红外光谱数据预测玉米样本中的水分、油脂、蛋白质和淀粉含量。实验采用一维卷积神经网络（1D-CNN）从复杂的光谱信号中提取特征，实现了高精度的定量分析。

---

## 📊 实验结果概览 (Test Set)

模型在测试集（24个样本）上表现优异，各项指标的均方误差（MSE）如下：

| 指标 (Component) | 水分 (Moisture) | 油脂 (Oil) | 蛋白质 (Protein) | 淀粉 (Starch) |
| :--- | :--- | :--- | :--- | :--- |
| **MSE** | 0.0270 | **0.0072** | 0.0825 | 0.2085 |

> **💡 结果分析：** 模型对油脂含量的预测最为精准（MSE最低），整体具备良好的泛化能力。

## 🛠️ 环境依赖

- **语言:** Python 3.8+
- **核心框架:** PyTorch
- **数据处理:** Pandas, NumPy, Scikit-learn
- **可视化:** Matplotlib, Seaborn

## 🧠 模型架构

采用 **1D-CNN (一维卷积神经网络)** 架构，专门针对光谱序列数据设计：

- **输入层:** 700维光谱向量 (1100nm-2498nm)。
- **卷积特征提取:** 3层 `Conv1d` + `BatchNorm` + `MaxPool`，自动捕捉光谱波峰波谷特征。
- **全连接回归:** 包含 Dropout (0.3) 的两层线性层，输出4种成分含量。
- **可复现性:** 代码中已固定随机种子 (`seed=42`)，确保实验结果可复现。

## 📂 数据集结构

- **样本总数:** 80个玉米样本。
- **光谱数据:** 700个波长通道。
- **标签数据:** 水分、油脂、蛋白质、淀粉（4列）。
- **划分比例:** 训练集 70% (56个), 测试集 30% (24个)。
